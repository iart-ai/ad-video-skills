# Ad Video Skills

> Claude Code skills for ad / advertising video — turn one motion-graphics template into batches of on-brand, A/B-testable ad creative, launch films, and testimonial clips.

## Install

```bash
npx skills add iart-ai/ad-video-skills
```

Works with Claude Code + 40+ agents.

## What's included

| Skill | What it does |
|---|---|
| [ad-creative-video](./ad-creative-video) | One data-driven ad template × a CSV = N message-matched variants, exported to every placement aspect for A/B testing. |
| [launch-video](./launch-video) | High-energy product launch / hype films: hook → tease → reveal → feature montage → end-card CTA, cut to the beat. |
| [testimonial-video](./testimonial-video) | Animated quote / review clips with staggered line reveals, key-phrase emphasis, fractional star ratings, and an author block. |

## When it activates

- You ask to make a video ad, launch film, hype reel, or animated testimonial.
- You want to batch-produce ad variants from a data table for A/B testing.
- You need one ad or film exported in multiple aspect ratios (9:16, 4:5, 1:1, 16:9) for Meta, TikTok, Reels, or YouTube.
- You want to turn a list of customer reviews into one social-proof video each.

## Example prompts

- "Make a UGC-style video ad and generate 5 hook variants from this CSV for A/B testing."
- "Build a 30s product launch video with a beat-synced reveal and an end-card CTA."
- "Turn this customer review into an animated testimonial with a 4.5-star rating."
- "Export this ad in 9:16, 4:5, and 1:1 for Meta and TikTok."
- "Take my list of reviews and render one quote video per review, brand-locked."
